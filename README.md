# 神経衰弱アプリByチームぬるぽ
## 概要
前期の授業の最終課題です。(制作はだいたい6～7月くらい)
授業で習ったの知識を使うのであれば自由に作っていいとの事で、チームぬるぽは神経衰弱アプリを作りました。チームの許可を得て個人アカウントで公開しています。
Pixel6aで動作確認テストをしました。 
トランプの配置は4×5の20枚
クラブのA、2～10
クローバーのA、2～10

## 機能の紹介
### 設定画面
制限時間(対人戦のみ)、プレイ人数(対人戦のみ)、EasyModeにするか選択できます。
EasyModeはクラブに青いフィルタがかかります。
リセットボタンで初期設定に戻します。
制限時間のデフォルトは5分00秒
人数のデフォルトは2人
デフォルトはEasyModeオフ
対人戦は1～4人で遊べます。
対人戦の制限時間は30秒から10分までです。
戻るは変更を保存せずタイトルに戻ります。
保存する場合は保存を押してください。

### 対人戦
ターンごとに交代して揃った数を競います。
揃うとその人のターンが延長されます。
制限時間を超えるもしくは全てのトランプを取り終わると順位が表示されます。
スマホを中断すると制限時間が止まります。画面をタップすると中断したときの残り時間とその人のターンから再開します。

### タイムアタック
制限時間はありません。1人専用モードです。
全てのトランプを取り終えるとクリアタイムと1～5番目に速くクリアしたタイムを表示します。
スマホを中断するとタイトルに戻ります。
 
### 最速記録画面
タイムアタックで1～5番目に速くクリアしたタイムを表示します。
NormalMode(通常モード)とEasyModeの記録を切り替えられます。
