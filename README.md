# 神経衰弱アプリByチームぬるぽ
## 概要
前期の授業の最終課題です。(制作はだいたい6～7月くらい)<br>
授業で習った知識を使うのであれば自由に作っていいとの事で、チームぬるぽは神経衰弱アプリを作りました。チームの許可を得て個人アカウントで公開しています。<br>
Pixel6aで動作確認テストをしました。 <br>
トランプの配置は4×5の20枚<br>
クラブのA、2～10<br>
ダイアのA、2～10<br>
背景にはAIが使われています。<br>

## 機能の紹介
### 設定画面
制限時間(対人戦のみ)、プレイ人数(対人戦のみ)、EasyModeにするか選択できます。<br>
EasyModeはクラブに青いフィルタがかかります。<br>
リセットボタンで初期設定に戻します。<br>
制限時間のデフォルトは5分00秒<br>
人数のデフォルトは2人<br>
デフォルトはEasyModeオフ<br>
対人戦は1～4人で遊べます。<br>
対人戦の制限時間は30秒から10分までです。<br>
戻るは変更を保存せずタイトルに戻ります。<br>
保存する場合は保存を押してください。<br>

### 対人戦
スマホを交代で手渡ししながらプレイします。<br>
ターンごとに交代して揃った数を競います。<br>
揃うとその人のターンが延長されます。<br>
制限時間を超えるもしくは全てのトランプを取り終わると順位が表示されます。<br>
スマホを中断すると残り時間が止まります。画面をタップすると中断したときの残り時間とその人のターンから再開します。<br>

### タイムアタック
制限時間はありません。1人専用モードです。<br>
代わりに経過時間が表示されます。<br>
全てのトランプを取り終えるとクリアタイムと1～5番目に速くクリアしたタイムを表示します。<br>
スマホを中断するとタイトルに戻ります。<br>
 
### 最速記録画面
タイムアタックで1～5番目に速くクリアしたタイムを表示します。<br>
NormalMode(通常モード)とEasyModeの記録を切り替えられます。<br>
